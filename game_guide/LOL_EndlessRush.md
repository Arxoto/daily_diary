# 无尽狂潮

## 锐雯
- 核心玩法
    - 被动 短距离高频率的伤害转换为护盾
    - 玩法 依靠装备打出高频伤害并转换为很厚的护盾 不需要点自带的技能及属性
- 必选的装备属性
    - UwU魔爆炮-技能急速  单体伤害高且属性增加技能频率
    - 效果范围            增加被动范围提高护盾
- 优选的装备属性
    - 回响蝠刃-飞弹数量  高频武器且增加技能数量
    - 漩涡手套-生命回复  高频武器且增加容错率
    - 冰爆护甲-护甲      增加容错率且属性变相增加护盾值
- 次选的装备属性
    - 战兔十字弩-暴击    贯穿伤害且属性不错
    - 回旋刃-移动速度    贯穿伤害且属性与自带技能适配
    - 伤害
    - 生命值
    - 经验
